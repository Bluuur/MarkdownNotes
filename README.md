# MarkdownNotes
+ 个人笔记,可能有错误
+ 推荐使用Typora打开:[typora.io](https://typora.io)
+ 使用Obsidian打开:[Obsidian](https://obsidian.md/)



>   ## 如何查看笔记
>
>   ### 方法1
>
>   点击右上角 `Code`, 选择 `Download ZIP` 下载到本地后解压查看
>   可以使用上述两个软件 (Typora/Obsidian)
>
>   ### 方法2
>
>   GitHub 在线查看
>
>   通过访问国外网站的方法 可以直接在 GitHub 显示图片
>   但不支持 $\LaTeX$ 公式渲染
>
>   ### 方法3
>
>   如果你熟悉 `git` 的使用
>   欢迎 `fork` 此仓库
