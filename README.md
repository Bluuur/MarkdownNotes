# MarkdownNotes
+ 个人笔记，可能有错误。
+ 推荐使用 [Typora](https://typora.io) 打开
+ 也可使用其他 Markdown 编辑器，如 [Obsidian](https://obsidian.md/) 。



>   ## 如何查看笔记
>
>   ### 方法1
>
>   下载本仓库到本地解压查看
>   可以使用上述两个软件 (Typora/Obsidian)
>
>   ### 方法2
>
>   GitHub 在线查看
>
>   可以直接在 GitHub 显示图片
>
