# 毛霉与根霉

## 分类

+  接合菌门毛霉属
+  接合菌门根霉属

## 营养体形态

+  菌丝无隔

## 无性生殖

+  不游动的孢囊孢子,孢子囊黑色或褐色,表面光滑

## 有性生殖

+  接合孢子 

## 生活环境/与人类关系

+  毛霉具有分解蛋白质和淀粉的能力,是用于制造腐乳/豆豉等食品及酿造业的重要菌种.也可引起蔬菜/水果发生腐败
+  根霉经常出现在淀粉质食品上,引起馒头/面包等贬值.分解淀粉能力强,在代谢过程中能产生淀粉酶和糖化酶,时酿造业中著名的生产菌种.
