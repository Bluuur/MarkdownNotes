# 控制语句

[toc]

## 条件语句

1. `if` 
2. `if...else...`
3. `if...elsif...else`
4. `unless...`
5. `unless...else...`
6. `unless...elsif...else...`
7. `switch`

## 三目运算符

```perl
Exp1 ? Exp2 : Exp3;
```

## 循环

1. `while`
2. `until`
3. `for`
4. `foreach`
5. `do...while`

## 循环控制语句

1. `next` "continue"
2. `last`  "break"
3. `continue` 将单次循环分隔为两个部分
4. `redo` 返回循环体的第一行重新执行，循环体中 `redo` 之后的部分不再执行。
5. `goto`
