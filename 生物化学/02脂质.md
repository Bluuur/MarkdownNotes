# 定义

含多重官能团的化合物，特点是疏水
脂肪酸与醇(主要是丙三醇)缩合形成脂质

脂质能够通过有机溶剂提取

+ 储存能量
+ 膜结构
+ 保护器官
+ 产生激素
+ 信号分子
