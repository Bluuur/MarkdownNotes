# 线性空间与线性变换

+ 矩阵
  + 矩阵是操作向量的工具
  + 矩阵代表线性变换
  + 物理
    + 有向线段
  + 计算机科学
    + 有序数表
  + 矩阵的逆表示对矩阵代表的线性变换进行「还原」
+ 行列式
  + 行列式的值 是变换前后空间中图形的面积, 物体的体积等变化的倍数
  + 值为零 表示降维
    + 可能降一维, 可能降多维
  + 值小于零 表示「翻转」
+ $P^{-1}AP$ 这样的形式表示由一个线性空间到另一个线性空间的「翻译」
+ 线性空间
  + 离散数学中的代数系统
  + 需要满足对「加法」与「数乘」运算的封闭
  + 还要满足 $8$ 种运算法则
  + 张成 $n$ 维空间 , 需要 $n$ 个基向量
+ 矩阵的特征值是在线性变换前后保持不变的向量
