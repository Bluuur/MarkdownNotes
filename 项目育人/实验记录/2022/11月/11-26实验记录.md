# 向样品中加入引物

## 样品

1. `NA 际 1`
2. `LB 际 2`
3. `10% LB 内 2`
4. `10% LB 际 1`

## 加入引物/mix

1. 每管样品加入 $F$ 引物 $1\ \mu L$ 

   1. 将 $96$ 种 $F$ 引物转移到 $12$ 个八连管中 `@yx`
   2. 用排枪加到样品管中

2. 配制 mix

   |       Item        |      Vol      |
   | :---------------: | :-----------: |
   |     $R$ 引物      | $100\ \mu L$  |
   |        mix        | $1000\ \mu L$ |
   | $\mathrm{ddH_2O}$ | $600\ \mu L$  |
   |      $total$      | $1700\ \mu L$ |

   + 样品与对应 $R$ 引物

     |     样品      | $R$ 引物 |
     | :-----------: | :------: |
     |   `NA 际 1`   |  $R\ 1$  |
     | `10% LB 内 2` |  $R\ 2$  |
     | `10% LB 际 1` |  $R\ 3$  |
     |   `LB 际 2`   |  $R\ 4$  |
