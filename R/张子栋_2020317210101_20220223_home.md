1. **运行命令 `x <- rnorm(n = 50, mean = 2, sd = 1.5)`, 对 `x` 运行 `summary` 函数, 解释结果中每一个函数返回值是什么统计量.**

   运行结果:

   ```R
   > x <- rnorm(n = 50, mean = 2, sd = 1.5)
   > summary(x)
      Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
    -1.184   1.424   2.156   2.188   2.899   4.713 
   ```

   + `rnorm()` 函数的参数:

     + `n` 产生随机数的个数

     + `mean` 均值

     + `sd` 标准差

     + > `r` 代表随机, `norm` 代表正态分布

   + `summary()` 函数的返回值:

     + `Min.` 最小值
     + `1st Qu` 上四分位数
     + `Median` 中位数
     + `Mean` 均值
     + `3Rd Qu.` 下四分位数
     + `Max` 最大值
