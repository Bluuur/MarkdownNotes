# HTML

Hyper Text Markup Language，超文本标记语言。HTML 语法松散，不区分大小写。

```html
<!--如果加上以下行 则使用的是 HTML5-->
<!DOCTYPE html>
<html>
	<head>
        <!--指定字符集-->
    	<meta charset='utf-8'>
        <title>网页标题</title>
    </head>
    <body>
        在这里输入内容，HTML 会忽略多个连续的空格为一个空格。也会忽略源代码中的换行符，
        如果需要换行，可以使用 <br> 标签，这是独目标签，不需要成对出现。<br>
        
    </body>
</html>

```
