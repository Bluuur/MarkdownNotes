# Redis键 key

+   `keys *` 查看当前库所有key (匹配: `keys *1`)
+   `set <key> <vlaue>` 添加key
+   `exist key` 判断某个key是否存在
+   `type key` 查看key的类型
+   `del key` 删除指定的key数据
+   `unlink key` 根据value选择非阻塞删除
    +   仅将keys从keyspace元数据中删除,真正的删除会在后续异步操作
+   `expire key 10` 为给定的key设置过期时间(10秒)
+   `ttl key` 查看还有多少秒过期
    +   `-1` 表示永不过期
    +   `-2` 表示已过期
+   `select [num]` 命令切换数据库
+   `dbsize` 查看当前数据库的key数量
+   `flushdb` 清空当前库
+   `flushall`  通杀全部库
